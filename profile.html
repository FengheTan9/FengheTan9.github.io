Research

I'm interested in medical image processing, computer vision and pattern recognition. Much of my research is about medical ultrasound image generation, segmentation, semi-supervised learning and distant domain transfer learning. Representative papers are highlighted.

arXiv

CMUNeXt: An Efficient Medical Image Segmentation Network based on Large Kernel and Skip Fusion
Fenghe Tang, Jianrui Ding, Lingtao Wang, Chunping Ning, S. Kevin Zhou.
arXiv, 2023
code / arXiv

In order to extract global context information while taking advantage of the inductive bias, we propose CMUNeXt, an efficient fully convolutional lightweight medical image segmentation network, which enables fast and accurate auxiliary diagnosis in real scene scenarios.

arXiv

Multi-Level Global Context Cross Consistency Model for Semi-Supervised Ultrasound Image Segmentation with Diffusion Model
Fenghe Tang, Jianrui Ding, Lingtao Wang, Min Xian, Chunping Ning.
arXiv, 2023
code / arXiv

We propose a Multi-level Global Context Cross-consistency (MGCC) framework that uses images generated by a Latent Diffusion Model (LDM) as unlabeled images for semi-supervised learning. Our approach enables the effective transfer of probability distribution knowledge to the segmentation network, resulting in improved segmentation performance.

Journal paper

Study on Financial Fraud Account Detection Based on Imbalanced Datasets
Fang Lu, Fenghe Tang, Junheng Huang, B Wang
Computer Engineering, 2021

The strategy uses the iForest algorithm for mixed sampling of the data to remove the majority of noisy data and reduce the difficulty of the classifier learning from the minor classes.On this basis, a random forest classifier is designed to implement the detection of the accounts involved in financial fraud.

Journal paper

Frequent path discovery algorithm for financial network
Fang Lu, Fenghe Tang, Junheng Huang, B Wang
Chinese Journal of Network and Information Security, 2019

A two-direction active edge searching method is proposed to solve the problem of evaluating the relationship strength between financial entities. And a breadth-first frequent path discovery algorithm with depth controlled is presented,with which the way how the financial flows is restored.
